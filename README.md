## ray tracing

### 1. 简介

> Rust 实现的简易光线追踪器



### 2. 结构



### 3. 依赖

+ [Rust](https://www.rust-lang.org/)
+ [rand](https://github.com/rust-random/rand)




### 4. 编译

```sh
make build
```



### 5. 运行

```sh
# 运行 in_one_weekend，指定文件名
# 文件必须是ppm文件格式
# 执行时间约30分钟
make run1 > image1.ppm
```

```sh
# 运行 the_next_week
make run2 > image2.ppm
```
